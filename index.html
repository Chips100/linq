Linq.js

A javascript implementation of .NETs LINQ enumerable interface with its extension methods.

Using Linq.js

var longSequence = linq.Enumerable.range(1, 100000000);
var projected = longSequence.skip(5).take(10)
                .where(function(x) { return x % 2 === 0; })
                .select(function(x) { return x * x; });

// Finishes in no time
console.log(projected.first());     // 36
console.log(projected.toArray());   // [36, 64, 100, 144, 196]

// Takes some seconds...
longSequence.toList();

Handling sequences in Javascript efficiently.

When working with lots of objects, adopting the easiest approach to process them can quickly cause performance issues. If you want to filter elements from a list of one million items that satisfy a certain condition and later on only need the first ten results, you do not want to check the condition for all items first just to throw the rest away. Chaining multiple transformations on a source sequence throughout different parts of the code to produce a result set is a common operation when working with data. Having an optimized way to create such constructs can simplify your code and speed up the development while preventing performance bottlenecks resulting from unnecessary operations. Linq.js provides an approach to tackle this problem that is easy to learn and adopt. It lets you build queries upon your source sequences that only start evaluating the items when it needs to.

Familiar syntax from .NET LINQ

If you have experience with the established .NET LINQ system and its extension methods, using Linq.js will feel natural to you. The method signatures are designed to closely match the .NET signatures while adhering to common Javascript conventions.

Getting started

Download and extract the zipped repository with the Download link on the top of this page. There are two Javascript files in the dist folder - an unminified version for development purposes and a minified version for production. Reference the appropriate file in your project to make use of the global linq object that serves as your entry point for making use of the library. Check the contents of the docs folder for a detailed documentation generated by jsdoc.
